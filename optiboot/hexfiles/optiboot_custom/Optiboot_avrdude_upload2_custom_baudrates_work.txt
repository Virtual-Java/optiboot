AFTER configure CUSTOM SPEED in Linux (Mint) using Linux-arbitrary-baud from Github:

git clone https://github.com/GrantEdwards/Linux-arbitrary-baud & cd Linux-arbitrary-baud
cc -g -Wall arbitrary-baud.c -o arbaud-Wall

./arbaud /dev/ttyACM0 125000

Test maximum (theoretical!!!) USB value (reported by the script!!!):

#./arbaud-Wall /dev/ttyACM0 2147483647
	actual speed reported 2147483647
#./arbaud-Wall /dev/ttyACM0 2147483648
	actual speed reported -2147483648

The maximum baud rate supported by Hoodloader2 (USB-Port of Arduino Uno, Mega, Leonardo, Micro) is 2 mega baud = 2 Mbaud = 2000 kbaud = 2000000 baud

The maximum baud rate of the native serial port of my computer was a bit higher than 115200 baud (e.g. to calibrate).
#./arbaud-Wall /dev/ttyS0 116352
	actual speed reported 116352
#./arbaud-Wall /dev/ttyS0 116353
	actual speed reported 115200
